TinTok

TinTok

TinTok is a cross-platform desktop baking timer application using Electron.js, HTML, CSS, and Node.js, designed with custom presets for different baking types. I created all of the visual components—including pixel-art backgrounds, buttons, and animated assets—while implementing a pixel-style UI with animated backgrounds and custom minimize/close controls. The app features image-based buttons and real-time countdown logic to deliver an intuitive, engaging user experience tailored for home bakers based on baking tin used.

Friends For Life

Friends For Life

Freinds For Life is a full-stack pet adoption web platform using Node.js, Express, EJS, HTML/CSS, and JavaScript, with persistent data handled through file I/O. The application allows users to register, log in, list pets for adoption, and search for available pets. It implementes dynamic form validation, session-based authentication, and a responsive, custom-styled frontend to ensure a smooth user experience. The platform is available via GitHub, complete with clear documentation and setup instructions for users.
